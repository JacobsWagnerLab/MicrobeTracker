loadimagestack

Syntax:

images = loadimagestack(filename);

images = loadimagestack(filename,dsp);

This function loads an image stack file into a 3D array.

  • <filename> - file name of a stack file, i.e. 'c:\test\test.stk'. The stack file must be a multipage TIFF file or BioFormats software must be installed (see Installation) to use other formats.

  • <images> - a 3D array of loaded images, where the third dimension is image number. The images are in 8-bit or 16-bit format depending on the bitdepth of the original files.

  • <dsp> - display progress bar if 1 (true), don't display if 0 (false) or absent.


See also: loadimageseries, saveimagestack, top pages: MicrobeTracker Tools, MicrobeTracker Suite.
