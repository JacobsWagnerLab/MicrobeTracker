integdic

Syntax:

integdic

integdic(angle)

integdic('detectmin')

integdic('disp')

This function converts a series of differential interference contrast (DIC) images into a symmetric format that resembles phase contrast images. The sourse images must be TIFF images placed into a single folder. The resulting images will have the same names as the sourse ones and will be placed into a folder of the same name with an added suffix '_processed', placed into the same parent forder.

  • <angle> - the direction of the main axis of the polarizer used to produce the DIC images relative to the camera (if known). If the value is not indicated, the function uses the default value of 135°. Though in many cases the default angle is correct, it assumes the default orientation of the camera.

  • 'detectmin' - detect the the direction of the main axis. This should typically be done is the default value gives unsatisfactory results.

  • 'disp' - display the original and the final images.
