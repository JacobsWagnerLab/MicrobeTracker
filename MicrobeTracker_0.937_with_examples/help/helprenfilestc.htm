renfilestc

Syntax:

renfiles

renfiles('multiple')

renfiles('copy')

This function renames/copies pairs/groups of files sorted by time into those sorted by filter block (the new folder names will be the file names in the original system, the file names will be the old folder names). It is designed for Nikon Elements software tiff output

It will prompt for the folder name containing the subfolders with images.

  • 'multiple' (optional) - will require selecting the parent folder and will process all subfolders.
  • 'copy' (optional) - copy the files instead of moving.
