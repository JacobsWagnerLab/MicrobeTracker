saveimageseries

Syntax:

images = saveimageseries(imagestack,filename);

images = saveimageseries(imagestack,filename,dsp);

This function saves a 3D array of images into a series of TIFF files.

  • <imagestack> - 3D array containing the images.
  • <filename> (optional) - name of the first file in the series of target files (TIFF by default, will open a file dialog if not provided). The series will be created by incrementing an ingex at the end or by incerting and incrementing such index if it is not provided.
  • <dsp> (optional) - display the progress bar if 1 (true), don't display if 0 (false) or absent.

See also: loadimagestack, saveimagestack, top pages: MicrobeTracker Tools, MicrobeTracker Suite.
