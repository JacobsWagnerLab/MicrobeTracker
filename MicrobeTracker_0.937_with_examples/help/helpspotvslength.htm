spotvslength

Syntax:

spotvslength(cellList)

spotvslength(cellList,pix2mu)

This function plots a digram of the number of spots vs. the length of a cell. The meshes and the spots have to be previously detected.

  • <cellList> - an array that contains cell meshes. You can drag and drop the file with the data into MATLAB workspace or open it using MATLAB Import Tool. The default name of the variable is cellList, but it can be renamed.
  • <pix2mu> - conversion factor from pixels to microns, the size of a pixel in microns.
